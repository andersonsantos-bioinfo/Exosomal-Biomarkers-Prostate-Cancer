# Exosomal-Biomarkers-Prostate-Cancer
Computational analysis pipeline for exosomal miRNA, lncRNA, and transcriptomic biomarkers in prostate cancer.
This repository contains analytical scripts and processed datasets related to the study of exosomal biomarkers in prostate cancer.

## Contents
- Transcriptomic analyses
- Differential expression analyses
- ROC and Precision-Recall analyses
- External validation datasets
- Processed data and supplementary materials

## Software
- RStudio 2026.04.0 Build 526 (Posit Software, PBC).
- Bioconductor packages
