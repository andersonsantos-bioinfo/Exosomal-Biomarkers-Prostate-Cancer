# Exosomal Biomarkers in Prostate Cancer

This repository contains analytical scripts and processed datasets related to transcriptomic analyses and exosomal biomarker discovery in prostate cancer using TCGA-PRAD cohorts.

## Contents

- TCGA-PRAD preprocessing
- Differential expression analyses
- DEG filtering
- Functional enrichment analyses
- Elastic Net modeling
- ROC and Precision-Recall analyses
- External validation cohorts

## Computational Environment

- R version 4.5.2
- Windows Server 2016 x64
- RStudio 2026.04.0 Build 526

## Main Packages

- DESeq2 1.50.2
- SummarizedExperiment 1.40.0
- GenomicRanges 1.62.1
- ggplot2 4.0.3
- dplyr 1.2.1

## TCGA Cohort

- 553 samples
- 501 tumor samples
- 52 normal samples
